# 🎬 Movie Streaming & Management Platform

A full-stack Movie Streaming & Management Platform built with **React.js, Tailwind CSS, Spring Boot, and MySQL**. The platform provides a modern and responsive interface for browsing movies while offering a powerful admin dashboard for managing movie content, media assets, and downloads. It also includes secure authentication, payment integration, and a dynamic movie download system.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Aman108-8/Movie.git
cd Movie
```

---

## Frontend Setup

```bash
cd frontend_movie
npm install
npm run dev
```

Frontend runs at:

```
http://localhost:5173
```

---

## Backend Setup

```bash
cd MovieBackend
mvn spring-boot:run
```

Backend runs at:

```
http://localhost:8080
```

---

# 🗄️ Database Configuration

Configure your database credentials in:

```
application.properties
```

Example:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/movie_db
spring.datasource.username=root
spring.datasource.password=your_password
```
