# 🎬 Movie Streaming & Management Platform

A full-stack movie streaming and management platform built using React, Tailwind CSS, Spring Boot, and MySQL. The project provides a modern UI for users and a powerful admin panel for managing movie content dynamically.

---

## 🚀 Features

### 👤 User Features
- Browse movies and series
- Responsive modern UI
- Dynamic movie pages
- Movie screenshots & quality selection
- Search functionality
- Fast loading experience

### 🔧 Admin Features
- Add, update, and delete movies
- Upload thumbnails, posters, and screenshots
- Manage movie qualities and links
- Dynamic content handling
- File upload management

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router

### Backend
- Spring Boot
- Spring MVC
- REST APIs
- MySQL
- Hibernate / JPA
